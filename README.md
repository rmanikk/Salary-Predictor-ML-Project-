# 💼 Salary Prediction Web App

A beginner-friendly Machine Learning web application that predicts salary based on user inputs like experience, age, education, and work type.

---

## 🚀 Features

* Predict salary using Machine Learning (Linear Regression)
* Clean and simple web interface using Flask
* Responsive design (works on mobile & desktop)
* Multiple input features:

  * Experience
  * Age
  * Education
  * Work Type (Remote / Onsite)

---

## 🧠 Machine Learning Model

* **Algorithm:** Linear Regression
* **Input Features:**

  * Experience
  * Age
  * Education (encoded)
  * Work Type (encoded)
* **Output:**

  * Predicted Salary
